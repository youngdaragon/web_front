This repository contains the implementation of this paper.
A convolutional neural network used to crop/weed semantic segmentation in multispectral images obtained from an unmanned aerial vehicle (UAV).

This is a multiclass classification task: the neural network predicts

each pixel in the multiespectral image is either a crop, weed or background.
The neural network structure is derived from the SegNet architecture, described in this paper.
